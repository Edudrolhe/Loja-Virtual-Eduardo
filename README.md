
# Loja Virtual em Next.js

Este projeto foi desenvolvido como parte de um trabalho acadêmico da disciplina de Desenvolvimento Web, aplicado para demonstrar conceitos modernos de front‑end, componentização, responsividade e integração com serviços externos.
Ele utiliza Next.js 14, React Context, CSS puro e integrações práticas como finalização de compra via WhatsApp.

Continuo vendendo alguns produtinhos por aqui =)

## Requisitos
- Node.js >= 18.17
